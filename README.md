# Salmonella in surface water
This repository contains R code developed for the predictive modeling for Salmonella complexity and for the pairwise serovar co-occurence analysis in the manuscript "Spatiotemporal characterization of Salmonella in surface water proximal to livestock production" by Esther G. Palmer, Jared C. Smith, Sarita Bugalia, Liliana C. M. Salvador, and Nikki W. Shariat. This project was funded by a USDA-NIFA award to NWS and LCMS (award # 2022-67017-36555).
